# 5. 功能清單

> 回上一層：[README.md](./README.md)

## 5.1 主管端（`/admin`）

- 登入頁
- 面試管理（模板列表 + 題目列表 tab + 面試連結 tab + 設定 tab）
- AI 產生題目
- 面試連結產生／重新生成／效期設定／使用統計
- 面試紀錄列表（統計卡、搜尋、篩選）
- 面試詳情頁（程式碼／題目／AI 對話／AI 評分／錄影播放／標記審閱）
- 應試者彙整頁
- AI 設定頁（全域預設模型與 prompt）
- 系統設定頁

## 5.2 候選人端（`/interview/[token]`）

- 進入頁（token 驗證、姓名填寫、螢幕分享授權、斷點續答載入）
- 作答頁（Monaco 編輯器、題目 Markdown 渲染、AI 對話區、自動存檔、計時器、錄影中指示）
- 提交流程（確認 dialog、停止錄影上傳、觸發評分與通知）
- 完成頁

## 5.3 後端 / 系統功能

- Token 驗證與狀態機（pending / in_progress / completed / expired）
- AI 產題、AI 對話、AI 自動評分（三種 Gemini 呼叫封裝）
- 螢幕錄影上傳與 Signed URL 播放
- Email 通知（非同步、失敗不阻塞）
- Rate limiting（AI 相關 API）
