# 9. 測試案例

> 回上一層：[README.md](./README.md) ／ 對應需求：[03-需求清單.md](./03-需求清單.md)、[12-追溯矩陣.md](./12-追溯矩陣.md)

| 編號 | 情境 | 前置條件 | 步驟 | 預期結果 | 對應需求 |
|---|---|---|---|---|---|
| TC-01 | 主管登入成功 | 已有主管帳號 | 輸入正確 Email/密碼登入 | 導向 `/admin/interviews`，可見側邊欄 | FR-01 |
| TC-02 | 主管登入失敗 | 無 | 輸入錯誤密碼 | 顯示錯誤訊息，停留登入頁 | FR-01 |
| TC-03 | AI 產題成功 | 已登入 | 輸入主題/難度/語言，點擊 AI 產生題目 | 回傳結構化題目草稿，可編輯後加入 | FR-03 |
| TC-04 | AI 產題逾額度 | 已登入，模擬 Gemini 429 | 點擊 AI 產生題目 | 顯示友善錯誤訊息，不顯示原始 API 錯誤 | FR-03, NFR-07 |
| TC-05 | 產生面試連結 | 已建立模板+題目 | 於連結 tab 設定 30 天效期並產生 | 產生唯一 token 連結，狀態為 pending | FR-05 |
| TC-06 | 重新生成連結使舊連結失效 | 已有現存連結 | 點擊「重新生成連結」 | 新連結可用，舊連結狀態變 expired 且無法進入 | FR-05 |
| TC-07 | 候選人以有效連結進入 | 連結狀態 pending | 開啟連結、填姓名、授權螢幕分享 | 建立 session，started_at 寫入，開始錄影 | FR-06 |
| TC-08 | 候選人以過期連結進入 | 連結 expires_at 已過 | 開啟連結 | 顯示「連結已過期」，無法進入作答 | FR-06, NFR-08 |
| TC-09 | 候選人以已完成連結重複進入 | session status=completed | 開啟同一連結 | 顯示「面試已完成」，拒絕再次作答 | FR-11 |
| TC-10 | 程式碼自動儲存 | 進行中 session | 編輯程式碼後等待 debounce 時間 | `final_answers`/`last_active_at` 於後端更新 | FR-07 |
| TC-11 | 斷點續答 | 已有 in_progress session 且已寫入部分程式碼與 AI 對話 | 關閉分頁後重新開啟同一連結 | 載入先前程式碼與對話，開始新一段錄影 | FR-08 |
| TC-12 | AI 助手一般問答 | 進行中 session | 輸入題目相關問題 | AI 回覆存入 `ai_messages`，互動次數 +1 | FR-09 |
| TC-13 | AI 助手提供完整解答 | 進行中 session，該面試 system prompt 允許 | 候選人明確要求解答 | AI 直接給出完整程式碼 | FR-09 |
| TC-14 | 螢幕分享中途取消 | 錄影進行中 | 候選人手動停止畫面分享 | 前端偵測 `track.onended`，提示重新授權 | FR-10 |
| TC-15 | 提交流程完整成功 | 已完成所有題目 | 點擊提交 → 確認 dialog → 確認 | 停止錄影並上傳、status=completed、觸發評分與通知、導向完成頁 | FR-10, FR-11, FR-12, FR-13 |
| TC-16 | Email 通知失敗不阻塞提交 | 模擬 Email 服務逾時 | 候選人提交 | 候選人仍看到「提交成功」完成頁，通知失敗僅記錄於後端 | NFR-06 |
| TC-17 | AI 自動評分寫入 | 候選人已提交 | 等待非同步評分完成 | `ai_score`／`ai_score_summary` 寫入並顯示於紀錄列表 | FR-12 |
| TC-18 | 主管檢視面試詳情 | 有已完成 session | 進入詳情頁 | 同版面顯示程式碼/題目/AI對話/評分/錄影 | FR-15 |
| TC-19 | 標記已審閱 | 詳情頁 review_status=pending | 點擊「標記為已審閱」 | 狀態變 reviewed，列表紅點數量減少 | FR-16 |
| TC-20 | RLS 權限隔離 | 兩位不同主管帳號 | 主管 A 嘗試存取主管 B 的 session 詳情 | 存取被拒絕/查無資料 | NFR-01, NFR-02 |
| TC-21 | 前端直接寫入被阻擋 | 候選人端 | 嘗試以 anon key 直接寫入 `interview_sessions` | 寫入被拒絕，須經後端 API | NFR-01 |
